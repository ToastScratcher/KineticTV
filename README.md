
# 📺 ToastTV | The Ultimate Web-Based IPTV Player

**ToastTV** is a lightweight, high-performance IPTV streaming application built for speed and simplicity. Designed to work flawlessly on mobile, tablet, and smart TV browsers (like LG webOS), it brings global television directly to your screen without the need for heavy apps or complex setups.

### ✨ Key Features
* **🌍 Global Categories:** Dedicated sections for **English 🏴󠁧󠁢󠁥󠁮󠁧󠁿**, **UK 🇬🇧**, **Freeview 📡**, **Kurdish ☀️**, **Arabic 🇮🇶**, and more.
* **🎬 Entertainment Hub:** Quick access to Movies, Sports, and Kids' content.
* **⭐ Favorites System:** Save your most-watched channels with a single tap for instant access on the home screen.
* **🔍 Smart Guide:** A real-time search engine built into the channel guide to find exactly what you want to watch.
* **🛠️ Resilient Streaming:** Built-in "Try Next" logic and error handling to skip offline streams automatically.
* **📱 Universal Compatibility:** Fully responsive design optimized for touchscreens and TV remotes alike.

### 🚀 How to Use
1. **Open the Link:** Simply visit your GitHub Pages URL in any modern web browser.
2. **Pick a Category:** Select your preferred language or genre from the home screen.
3. **Watch & Enjoy:** Tap a channel to start the live stream. Use the **☰ GUIDE** button to switch channels or search the list.
